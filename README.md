## 1.0.0

- 翻译工具，快捷翻译短句，支持页面划词翻译
- 快捷搜索，选中文本后右键菜单添加快捷的百度搜索、必应搜索、谷歌搜索
- 网页自动滚动，自动阅读
- 选中复制，适用于限制了用户复制功能的网页，如百度文库等网站
- 网页图片匣子，查看自浏览器启动浏览过的所有图片
- 全网VIP视频去广告免费看，支持爱奇艺、优酷、腾讯视频、bilibili
- 网页版抖音添加下载按钮，点击直接下载视频
- MusicTools 无需输入暗号直接打开下载页
